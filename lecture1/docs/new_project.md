# New Project Guide

> 이 파일은 플레이스홀더입니다. 실제 신규 프로젝트 가이드 문서로 교체해주세요.
> 원본 URL: https://raw.githubusercontent.com/hw5511/vibe-web/main/docs/new_project.md

## 새 프로젝트 시작하기

1. `_template_settings` 디렉토리를 복사
2. 프로젝트 이름으로 폴더 변경
3. `package.json`의 `name` 필드 수정
4. `npm install` 실행
5. `npm run dev`로 개발 서버 시작

## 포함된 패키지
- React 18
- React Router 7
- MUI 9
- Emotion
- Fontsource Roboto
