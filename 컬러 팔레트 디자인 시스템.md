:root {

  --color-button-text:    #F7F3EF;  /* 버튼 위 텍스트 */
  --color-link:           #8B1A2F;  /* 링크 기본 색상 */
  --color-link-hover:     #6B1D35;  /* 링크 hover 색상 */
  --color-focus-ring:     rgba(139, 26, 47, 0.4); /* 포커스 링 (접근성) */

  /* ── Status Colors ── */
  --color-success:        #4A7C59;  /* 성공: 그린 계열 (뉴트럴 톤 맞춤) */
  --color-warning:        #B8860B;  /* 경고: 골드/머스터드 */

## 컬러 사용 가이드

### Primary Color (`#1C0A14`) — 딥 버건디

| 사용처 | 예시 |
|--------|------|
| 헤더/네비게이션 배경 | `background-color: var(--color-primary)` |
| 히어로 섹션 배경 | 전체 폭 배너, 랜딩 영역 |
| 푸터 배경 | 페이지 하단 브랜드 컬러 강조 |
| 로고/워드마크 | 브랜드 아이덴티티 요소 |

```css
/* 사용 예시 */
header {
  background-color: var(--color-primary);
  color: var(--color-text-inverse);
}
```

---

### Secondary Color (`#8B1A2F`) — 크림슨 레드

| 사용처 | 예시 |
|--------|------|
| CTA 버튼 | "더 보기", "구매하기" 버튼 |
| 강조 텍스트 | 가격, 할인율, 뱃지 |
| 링크 | 텍스트 링크, 인라인 강조 |
| 섹션 제목 하이라이트 | 언더라인, 좌측 보더 |

```css
/* 사용 예시 */
.btn-primary {
  background-color: var(--color-button-primary);
  color: var(--color-button-text);
  box-shadow: var(--shadow-button);
}
.btn-primary:hover {
  background-color: var(--color-button-hover);
}

.price {
  color: var(--color-text-price);
  font-weight: 700;
}
```

---

### Background Colors — 아이보리/크림 계열

| 변수 | Hex | 사용처 |
|------|-----|--------|
| `--color-bg-primary` | `#F2EDE8` | 페이지 기본 배경 |
| `--color-bg-secondary` | `#EDE7E1` | 대안 섹션 배경, 줄무늬 레이아웃 |
| `--color-bg-card` | `#F7F3EF` | 카드 컴포넌트, 모달 배경 |

```css
/* 교차 섹션 배경 패턴 */
section:nth-child(even) {
  background-color: var(--color-bg-secondary);
}
section:nth-child(odd) {
  background-color: var(--color-bg-primary);
}
```

---

### Text Colors

| 변수 | 사용 상황 |
|------|-----------|
| `--color-text-primary` (`#1A1A1A`) | h1~h3, 카드 제목, 핵심 정보 |
| `--color-text-secondary` (`#555555`) | 본문 단락, 설명 텍스트 |
| `--color-text-muted` (`#888888`) | 날짜, 메타 정보, placeholder |
| `--color-text-inverse` (`#F2EDE8`) | 어두운 배경(primary) 위 텍스트 |

```css
/* 타이포그래피 계층 */
h1, h2 { color: var(--color-text-primary); }
p       { color: var(--color-text-secondary); }
.meta   { color: var(--color-text-muted); }
```

---

### Interactive Colors

```css
/* 버튼 컴포넌트 */
.btn {
  background-color: var(--color-button-primary);
  color: var(--color-button-text);
  transition: background-color 0.2s ease, box-shadow 0.2s ease;
}
.btn:hover  { background-color: var(--color-button-hover); }
.btn:focus  { outline: 2px solid var(--color-focus-ring); outline-offset: 2px; }

/* 링크 */
a {
  color: var(--color-link);
  text-decoration: none;
}
a:hover { color: var(--color-link-hover); text-decoration: underline; }
```

## 반응형 고려사항

### 다크모드 (prefers-color-scheme)

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경: 아이보리 → 딥 다크로 전환 */
    --color-bg-primary:   #1A1015;
    --color-bg-secondary: #21141B;
    --color-bg-card:      #2A1A22;

    /* 텍스트: 어두운 배경에 맞게 밝게 */
    --color-text-primary:   #F0EAE5;
    --color-text-secondary: #C4B8B0;
    --color-text-muted:     #8A7E78;

    /* 테두리: 어두운 배경에 맞게 조정 */
    --color-border:       #3D2B35;
    --color-border-light: #2E1E28;

    /* Primary/Secondary 는 그대로 유지 (이미 어두운 색상) */
    /* --color-primary, --color-secondary 변경 없음 */

    /* 그림자: 다크 모드에서 더 강하게 */
    --shadow-card:   0 2px 8px rgba(0, 0, 0, 0.3);
    --shadow-modal:  0 8px 32px rgba(0, 0, 0, 0.5);
  }
}
```

## 접근성 체크리스트

| 조합 | 대비율 | WCAG 등급 | 용도 |
|------|--------|-----------|------|
| `#1A1A1A` on `#F2EDE8` | 14.9:1 | AAA | 본문 텍스트 |
| `#8B1A2F` on `#F2EDE8` | 7.2:1  | AAA | 링크, 강조 |
| `#F2EDE8` on `#1C0A14` | 12.1:1 | AAA | 헤더 텍스트 |
| `#555555` on `#F2EDE8` | 6.1:1  | AA  | 서브 텍스트 |
| `#888888` on `#F2EDE8` | 3.5:1  | AA (Large) | 작은 텍스트 주의 |

> WCAG 2.1 기준: 일반 텍스트 AA = 4.5:1, AAA = 7:1 / 큰 텍스트 AA = 3:1
